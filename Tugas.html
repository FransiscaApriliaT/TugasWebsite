RANGKUMAN

HTML merupakan sebuah bahasa markup yang digunakan untuk membuat sebuah halaman web dan menampilkan berbagai informasi di dalam sebuah browser. HTML saat ini merupakan standar Internet yang didefinisikan dan dikendalikan penggunaannya oleh World Wide Web Consortium (W3C).

Sejarah HTML dimulai dari HTML 2.0 (RFC1866) disetujui sebagaistandard pada 22 September 1995, HTML 3.2 pada 14 Januari 1996, HTML 4.0 pada 18 Desember 1997, ISO/IEC 15445:2000 “ISO HTML” berdasar padaHTML 4.01 Strict → 15 Mei 2000 XHTML 1.0 → gabungan HTML & XML 24 Januari 2000, XHTML 1.1 pada 16 February 2007.

Ciri – ciri Dokumen HTML

• Ekstensi file berupa .htm atau .html
• Non case sensitive.
• Terdiri atas tag - tag pembuka dan penutup (walaupun ada beberapa tag yang tidak memiliki penutup).
• Tag - tag saling berpasangan & bersarang.

Tag HTML Dasar

Tag Heading

Tag heading digunakan untuk menuliskan judul dan sub-judul. Contoh:

<h1>Judul Utama</h1>

Tag Paragraf, HR, dan BR

  • <p>: Membuat paragraf baru
  • <br />: Break-line, berpindah ke baris selanjutnya
  • <hr />: Horizontal-line, menambahkan garis horizontal

Tag Font

Tag ini digunakan untuk mengatur font (jenis, ukuran, warna) pada halaman web. Contoh:

<font color="blue" size="7" face="Arial">Contoh Text</font>

Tag Gambar

Tag <img> digunakan untuk menyisipkan gambar. Contoh:

<img src="logo.jpg" width="100" height="100" alt="Logo" />

Tag Anchor/Hyperlink

Tag <a> digunakan untuk membuat hyperlink ke halaman lain. Contoh:

<a href="http://example.com">Kunjungi Website</a>

Tag Komentar

Tag komentar digunakan untuk memberikan catatan yang tidak akan tampil di browser. Contoh:

<!-- Ini adalah komentar -->
